## Description
> This project returns the user a list of mock employees for a company. You can then filter through employees using any of the criteria included on the table, or sort the employees in alphabetical order using their first name. This project was created through use of React.js and bootstrap

## Installation
> To install this program, you can clone the repository to your local machine. First run an npm install in the directory where you cloned the app to install the necessary packages. Next, you can run npm start to run the app on localhost:3000! Alternatively, you can view this website on github pages by navigating to the following link: https://nickkdb.github.io/EmployeeDirectory/

## Usage
> To use the program, navigate to the deployed version of the app using the link above or run the app locally. Once on the page, try filtering through the employees by searching for different criteria!
